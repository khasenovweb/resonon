Бортовая система дистанционного зондирования

Воздушные гиперспектральные датчики Resonon представляют собой полностью интегрированные решения «под ключ» со всем аппаратным и программным обеспечением, необходимым для получения георегистрируемых гиперспектральных данных.

Системы могут быть оснащены любой из гиперспектральных камер Resonon , охватывающих спектральный диапазон 350–1700 нм.

Бортовые системы могут быть установлены как на пилотируемых, так и на беспилотных летательных аппаратах. Масса полезной нагрузки начинается от 1,55 кг. Свяжитесь с нами, чтобы обсудить интеграцию с вашим дроном или платформой дистанционного зондирования.

Системные компоненты:

  • Гиперспектральная камера
  • Объектив
  • Блок управления системой - для сбора данных на борту
  • 600 ГБ встроенного хранилища данных
  • GPS / IMU - для записи положения и ориентации датчика
  • Программное обеспечение для георекламы - для корректировки информации GPS / IMU

Датчик освещенности нисходящего потока Точечный спектрометр, ориентированный вверх, для контроля солнечного освещения во время полета.
Калибровочные грунтовые брезенты Тканевые брезенты с известной отражательной способностью, используемые в качестве калибровочного стандарта. Размещено в интересующем месте.
Радиометрическая калибровка Лабораторное измерение абсолютного отклика тепловизора. Используется во время постобработки данных для преобразования необработанных данных в сияние.
