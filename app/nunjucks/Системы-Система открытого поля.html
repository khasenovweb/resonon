Портативная система гиперспектральной визуализации

Наша наружная система используется для получения гиперспектральных данных в тех областях, где мобильность является критически важной. Гиперспектральная камера Resonon с нажимными щетками устанавливается на поворотный столик на специально модифицированном штативе. Во время сбора данных сцена вращает гиперспектральную камеру, просматривая интересующую сцену. Тепловизор и столик управляются с помощью программного обеспечения Spectronon.

Необработанные гиперспектральные данные преобразуются в яркость и отражательную способность с использованием данных радиометрической калибровки и калибровочного стандарта, помещенного в сцену. Эталон калибровки представляет собой специальный брезент с известной (в основном плоской) отражательной способностью. Эти операции можно выполнить с помощью программного обеспечения Spectronon.

Системы Resonon - это готовые к немедленному использованию решения под ключ, включая все оборудование и программное обеспечение, необходимое для сбора и анализа гиперспектральных данных.

Наружные системы могут быть оснащены любой гиперспектральной камерой Resonon, охватывающей спектральный диапазон 350–1700 нм.

Системные компоненты:
  • Гиперспектральная камера
  • Объектив
  • Блок управления системой - для сбора данных на борту
  • 600 ГБ встроенного хранилища данных
  • GPS / IMU - для записи положения и ориентации датчика
  • Программное обеспечение для георекламы - для корректировки информации GPS / IMU
